## Your complete SOP reference library

The **SOP Library** provides a centralized view of all your practice's standard operating procedures. SOPs are organized by category, making it easy to find the right procedure when you need it.

Access the SOP Library by navigating to **Knowledge** in the sidebar and selecting the **SOP Library** tab, or by navigating directly to the legacy SOP Library page.

<!-- Screenshot needed: SOP Library page showing the category grid with several categories and SOP counts -->

## Page header

The SOP Library page displays:

- **Title:** "SOP Library"
- **Subtitle:** "Your complete reference library of standard operating procedures"
- **Create New SOP** button -- Opens the [SOP creation wizard](/knowledge/creating-sops)
- **Ask SOP Assistant** button -- Opens the [SOP Assistant](/knowledge/sop-assistant)
- **Search bar** -- Search SOPs by title with keyboard shortcut support

## Searching SOPs

The search bar at the top of the page lets you find SOPs quickly by title. Type in the field labeled **Search SOPs by title...** to filter results in real time.

You can also press **Cmd+K** (Mac) or **Ctrl+K** (Windows) to focus the search field from anywhere on the page.

When searching:

- Results appear immediately as you type, showing the number of matches (e.g., "3 results for 'autoclave'").
- Each result shows the SOP title and its assigned category.
- Select any result to open the SOP detail page.
- Select **Clear search** to reset and return to the category grid view.

If no SOPs match your search, a message reads "No SOPs found matching" followed by your search term.

## Category grid

When not searching, SOPs are organized into a category grid. Each category card displays:

- **Category name** (e.g., "Sterilization", "Patient Care")
- **Category icon** -- A visual identifier for the category
- **SOP count** -- The number of SOPs assigned to that category

Select a category card to drill into it and see all SOPs within that category.

### Managing categories

Select the **Manage** button (gear icon) in the Categories section header to open the category settings dialog. From there you can:

- Create new categories
- Rename existing categories
- Change category icons
- Delete empty categories

## Category detail view

When you select a category, the page transitions to a detail view showing all SOPs in that category. This view includes:

- The category name as a heading
- A **Back** button to return to the category grid
- A list of all SOPs in the category
- Options to delete SOPs from the category

Select any SOP in the list to navigate to its full detail page where you can read and edit the procedure.

## Unassigned SOPs

Below the category grid, an **Unassigned SOPs** section appears if any SOPs have not been assigned to a category. From this section you can:

- View all uncategorized SOPs
- Move SOPs into a category using the move action

<Callout kind="tip">
Keep your SOP library organized by assigning all SOPs to a category. This makes it easier for your team to find the right procedure quickly.
</Callout>

## Empty state

If you have not created any SOPs yet, the page displays an onboarding state that guides you through creating your first SOP. Select the prompt to open the [SOP creation wizard](/knowledge/creating-sops).
