## Structured learning for your team

**Training Paths** are curated sequences of SOPs that guide your team through structured learning. Use them to onboard new hires, train staff on new procedures, or ensure everyone follows the same protocols.

<Callout kind="alert">
Training path creation and management is a **Professional** tier feature. Essential tier users can view training paths but cannot create or edit them. Upgrade to Professional to unlock this capability.
</Callout>

Navigate to **Knowledge** in the sidebar. The **Training Paths** tab is shown by default.

## Browsing training paths

The Training Paths page displays your paths in a visually organized layout:

- **Hero card** -- The most recent (or featured) training path in each category is displayed as a larger hero card with prominent details.
- **Compact cards** -- Additional paths in each category are shown in a two-column grid of smaller cards.

Each training path card displays:

- **Path name** (e.g., "New Hire Orientation")
- **Path icon** -- One of 11 available icons (book, graduation cap, star, shield, heart, alert, wrench, clipboard, timer, monitor, growth)
- **Description** -- A brief summary of the path
- **SOP count** -- How many SOPs are included
- **Difficulty level** badge (Beginner, Intermediate, or Advanced) when set
- **Required** badge when the path is marked as mandatory

Select any training path card to open its [detail page](#training-path-detail-page).

## Categories

Training paths can be organized into categories. When categories exist, paths are grouped under category headers with:

- **Category icon** -- A visual identifier
- **Category name** (e.g., "Foundational Training")
- **Path count** badge (e.g., "3 paths")
- **Description** -- An optional subtitle for the category

Paths not assigned to any category appear in an **Uncategorized** section at the bottom.

### Managing categories

Hover over a category header to reveal edit and delete buttons:

- **Edit** (pencil icon) -- Opens an inline editor where you can change the category name, icon, description, and featured path.
- **Delete** (trash icon, admin only) -- Removes the category. Paths in the deleted category become uncategorized.

You can also create new categories by selecting **Create Category** in the page header or using the **Create Category** button at the bottom of the category list.

## Training path detail page

Select a training path to open its detail page. This page shows comprehensive information about the path and its SOP sequence.

{/* Screenshot needed: Training path detail page showing the header, info cards, and SOP list */}

### Header

The detail page header displays:

- **Path icon** -- Large icon in a colored container
- **Path name** -- The title of the training path
- **Required** badge (if applicable)
- **Difficulty** badge (Beginner, Intermediate, or Advanced)
- **Description** -- The full description text
- **SOP count** -- e.g., "5 SOPs"
- **Estimated time** -- Calculated at approximately 10 minutes per SOP (e.g., "~50 min total")

A **Back to Training Paths** button returns you to the Training Paths list.

### Detail info cards

When goals, target audience, or difficulty level are set, info cards appear below the header in a three-column grid:

- **Goals** card -- What trainees will achieve after completing the path
- **Target Audience** card -- Who the path is designed for (e.g., "New dental assistants")
- **Difficulty** card -- The difficulty level (Beginner, Intermediate, or Advanced)

### SOP list

Below the detail cards, the SOP list displays all procedures in the training path in their assigned order. Each SOP entry shows its title and can be selected to navigate to the full SOP detail page.

### Admin actions

Users with admin or manager roles see additional controls:

- **Add SOP** -- Navigate to the [Add SOP page](/knowledge/creating-training-paths#adding-sops-to-a-training-path) to add new or existing SOPs to the path.
- **Edit** -- Toggle edit mode to reorder or remove SOPs from the path.
- **Edit Details** (in edit mode) -- Open a dialog to update the path name, description, icon, goals, target audience, difficulty level, and required status.
- **Done** -- Exit edit mode.
- **Delete** (trash icon) -- Permanently delete the training path. A confirmation dialog appears before deletion.

## Templates

ChairPulse provides pre-built training path templates to help you get started quickly. Select **Browse Templates** from the Training Paths page header to open the template browser.

Templates include pre-configured names, descriptions, icons, goals, target audiences, and difficulty levels. Selecting a template pre-fills the creation dialog so you can customize and create the path quickly.

<Callout kind="tip">
The Browse Templates button only appears after you have created at least one training path. For your very first path, use the **Create Path** button and configure it manually or follow the onboarding prompts.
</Callout>

## Empty state

If no training paths exist, an onboarding state guides you through creating your first path. Professional tier users see a **Create Path** button and an option to explore templates. Essential tier users see an upgrade prompt.
